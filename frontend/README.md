# CenterBack.AI Frontend

Next.js dashboard for CenterBack.AI (Network Intrusion Detection demo).

## Setup

```bash
cd frontend
npm install
npm run dev
```

By default, the UI calls the backend at `http://localhost:8000`. Override with:

```bash
NEXT_PUBLIC_API_URL=http://localhost:8000 npm run dev
```

## Scripts

- `npm run dev` - start dev server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - ESLint
